Submit

1. Visitcs50.me, log in with your GitHub account, and click Authorize submit50. You should receive two emails from GitHub inviting you to join CS50’s “organizations” on GitHub. Click the button in those emails to accept both of the invitations.

2. Install Git and, optionally,install submit50.

3. Using Git, push your work to https://github.com/submit50/USERNAME.git, where USERNAME is your GitHub username, on a branch called cs50/web/2018/x/projects/0 or, if you’ve installed submit50, execute.

4. submit50 cs50/web/2018/x/projects/0

instead.

5. Record a 1- to 5-minute screencast in which you demonstrate your app’s functionality and/or walk viewers through your code. Upload that video to YouTube (as unlisted or public, but not private) or somewhere else. Submit this form.
